# Izel Language Specification

## Introduction
Izel is a systems programming language designed for ultimate safety, predictability, and performance. This specification defines the syntax, semantics, and standard library of the language.

### Core Philosophy
1.  **Predictability Over Convenience**: No hidden allocations or behaviors.
2.  **The Safety Triad**: Effects, Witnesses, and Contracts ensure correctness.
3.  **Modern Metaprogramming**: Integrated compile-time execution and procedural macros.
